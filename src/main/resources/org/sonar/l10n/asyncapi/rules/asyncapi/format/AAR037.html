Noncompliant Code Example (AsyncAPI 2)

asyncapi: '2.4.0'
info:
  title: Example API
  version: '1.0.0'
servers:
  production:
    url: mqtt://broker.example.com
    protocol: mqtt
channels:
  /example-channel:
    description: Example channel for testing.
    bindings: # Noncompliant {{AAR037: You must specify the version of the binding}}
      mqtt:
        qos: 1
    subscribe:
      summary: Example subscription
      operationId: exampleSubscription
      message:
        contentType: application/json

Compliant Solution (AsyncAPI 2)

    asyncapi: '2.4.0'
    info:
      title: Example API
      version: '1.0.0'
    servers:
      production:
        url: mqtt://broker.example.com
        protocol: mqtt
    channels:
      /example-channel:
        description: Example channel for testing.
        bindings:
          mqtt:
            qos: 1
            version: 1.0.0
        subscribe:
          summary: Example subscription
          operationId: exampleSubscription
          message:
            contentType: application/json
