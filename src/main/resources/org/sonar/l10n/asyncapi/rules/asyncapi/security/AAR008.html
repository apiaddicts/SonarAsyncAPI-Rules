The servers should be defined

Noncompliant Code Example (AsyncAPI 2)

  # Noncompliant {{AAR008: The servers should be defined}}
  asyncapi: '2.4.0'
  info:
    title: Swagger Petstore
    version: '1.0.0'
  channels:
    /pets: {}

Compliant Solution (AsyncAPI 2)

  asyncapi: '2.4.0'
  info:
    title: Swagger Petstore
    version: '1.0.0'
  servers:
    default:
      url: 'api.example.com'
      protocol: 'https' 
      description: Default server for the pet store
  channels:
    /pets: {}
