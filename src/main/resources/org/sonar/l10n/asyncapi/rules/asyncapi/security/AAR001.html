For security reasons, the HTTPS protocol is mandatory.

Noncompliant Code Example (AsyncAPI 2)

  asyncapi: '2.4.0'
  info:
    title: Swagger Petstore
    version: '1.0.0'
  servers:
    default:
      url: 'api.example.com'
      protocol: 'http' # Noncompliant {{AAR001: Protocol https is mandatory}}
      description: Default server for the pet store
  channels:
    /pets: {}

Compliant Solution (AsyncAPI 2)

  asyncapi: '2.4.0'
  info:
    title: Swagger Petstore
    version: '1.0.0'
  servers:
    default:
      url: 'api.example.com'
      protocol: 'https' 
      description: Default server for the pet store
  channels:
    /pets: {}
