Por razones de seguridad, el protocolo HTTPS es obligatorio.

Ejemplo de código no compatible (AsyncAPI 2)

  asyncapi: '2.4.0'
  info:
    title: Swagger Petstore
    version: '1.0.0'
  servers:
    default:
      url: 'api.example.com'
      protocol: 'http' # Noncompliant {{AAR001: El protocolo https es obligatorio}}
      description: Servidor por defecto para la tienda de mascotas
  channels:
    /pets: {}

Solución compatible (AsyncAPI 2)

  asyncapi: '2.4.0'
  info:
    title: Swagger Petstore
    version: '1.0.0'
  servers:
    default:
      url: 'api.example.com'
      protocol: 'https' 
      description: Servidor por defecto para la tienda de mascotas
  channels:
    /pets: {}
